✈️ Travel Recommendation API

A simple and scalable Travel Recommendation REST API built with .NET 10, Entity Framework Core, and SQL Server (Docker).

This project was created to demonstrate backend development skills such as REST API design, CRUD operations, DTO usage, mapping, database migrations and clean project structure.

🚀 Features
▬ RESTful API architecture
▬ Full CRUD operations
▬ SQL Server running on Docker
▬ Entity Framework Core + Migrations
▬ DTO pattern (Create / Update / Response DTOs)
▬ Mapster object mapping
▬ Swagger / OpenAPI documentation
▬ Clean layered structure (Controller → Service → Data)

🧱 Tech Stack
▬ .NET 10 Web API
▬ Entity Framework Core
▬ SQL Server (Docker container)
▬ Mapster
▬ Swagger / OpenAPI
Git & GitHub


📖 API Endpoints
Method	          Endpoint	                         Description
GET	        /api/recommendation	              Get all recommendations
GET	        /api/recommendation/{id}	               Get by id
POST	      /api/recommendation	                       Create
PUT      	  /api/recommendation/{id}	                 Update
DELETE	    /api/recommendation/{id}	                 Delete
